# Bookmarks application

Social bookmarking application.

It sorts bookmarks by click history.

```bookmarks
bookmarks:
  - name: foo
    link: 'http://example.com/foo'
    clicks:
      - - 20180123
        - 2
  - name: bar
    link: 'http://example.com/bar'
    clicks:
      - - 20180122
        - 1
      - - 20180123
        - 3
```
